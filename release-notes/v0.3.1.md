# Codex Token Bar v0.3.1

This release polishes the first-run experience, update path, and the live dashboard details.

## Highlights

- New first-run setup guide: confirms the Codex directory, recommends launch-at-login, and lets you enable automatic update checks.
- New menu bar control: open the main window, check for updates, toggle automatic update checks, toggle launch-at-login, or quit from the macOS status bar.
- Launch-at-login now starts quietly after setup, keeping the floating/status token meter available without opening the full dashboard.
- Account quota display now shows a retry hint when quota reading fails, so transient network/API failures are easier to recover from.
- The recent 24-hour chart hover card now appears only while hovering and floats above the curve without clipping at the edges.
- The activity mode selector and README screenshots were refreshed to show cache hit rate, quota lines, and the new floating panel layout.
- Release packaging keeps the styled DMG background and the Sparkle update signature checks from the v0.3.0 release line.

## Update Notes

- Sparkle checks the GitHub appcast when automatic update checks are enabled, or when you choose `Codex Token Bar -> Check for Updates...`.
- Updates are still user-confirmed; the app does not silently install a new version in the background.
- The app remains ad-hoc signed and is not Apple notarized. Verify the SHA256 checksum before opening if macOS warns about an unidentified developer.
