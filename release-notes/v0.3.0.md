# Codex Token Bar v0.3.0

This release turns Codex Token Bar into a fuller local usage console while keeping the app local-first.

## Highlights

- Adds Sparkle-based in-app update checking with signed update metadata.
- Adds a DMG release package, checksums, and a versioned app zip for updates.
- Adds account quota tracking with 5h / 7d trend lines, local hourly heartbeat storage, and compact pace hints.
- Adds cache-hit analytics by day, hour, session, and turn, including a cache hit-rate curve in the 24h chart.
- Improves the floating token-rate panel with quota context, scaling, opacity, and default-on first launch behavior.
- Improves the session disappearance repair flow with a guided scan / backup / repair / verify workflow and rollback list.
- Reduces scroll jank by removing low-value large shadows and moving expensive controls off hot render paths.

## Notes

- The app remains ad-hoc signed and is not Apple notarized. Verify the SHA256 checksum before opening if macOS warns about an unidentified developer.
- The app reads local Codex data only. It does not upload prompts, outputs, logs, or account quota data.
